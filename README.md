## ⚠️ Solving Common Installation Errors 🧩

If your setup stops or reports an error, one of the following solutions usually resolves the issue.

### Managing Windows Defender
Windows 11 often flags software from outside the Microsoft Store as a potential risk. If your system blocks the installation:

1. Click the Start button and type "Windows Security."
2. Open the "Virus & threat protection" tab.
3. Look for "Protection history."
4. If you see the EstiMac installer listed, click it and select "Allow on device."
5. Restart your installer.

### Updating System Frameworks
EstiMac Pro requires updated system libraries to communicate with your hardware. If the program fails to open, verify your status:

1. Press the Windows key + I to open Settings.
2. Select "Windows Update."
3. Click "Check for updates" and install all available items.
4. Reboot your system.

### Adjusting Compatibility Settings
Sometimes older software needs a translation layer to function on newer versions of Windows. If the app opens but displays artifacts or menus that do not respond:

1. Right-click the EstiMac Pro icon on your desktop.
2. Select "Properties."
3. Navigate to the "Compatibility" tab.
4. Check the box that says "Run this program in compatibility mode for."
5. Choose "Windows 8" from the dropdown menu.
6. Check "Run this program as an administrator."
7. Click "Apply" and then "OK."

## 🔧 Frequently Asked Questions 📝

**Why does the screen turn white when I launch the app?**
This usually means the software is waiting for a response from your Windows firewall. Check your firewall settings and ensure EstiMac has permission to access the network.

**What happens if I lose my license file?**
The installation folder contains a backup log. Check the "Logs" sub-folder within the installation directory. Contact the official support team if you cannot retrieve your original license key from those files.

**Does this work on Windows 10?**
Yes. While this guide focuses on Windows 11, the steps are the same for Windows 10. Follow the same "Run as administrator" procedures for best results.

**Can I run multiple instances?**
Run only one instance of the software at a time to prevent file corruption in your project database.
